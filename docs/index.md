## Aura Alpha 是什么？

Aura Alpha 是一款基于 **ROS2 Humble** 的智能球形机器人，搭载地平线 **RDK X5** 计算平台（10 TOPS BPU），具备：

| 能力 | 说明 |
|------|------|
| **视觉感知** | 实时人体检测与跟踪，BPU 硬件加速 |
| **语音交互** | 支持火山引擎、百度等云端大模型 |
| **自主运动** | 双轮差速驱动，跟随/避障行为 |
| **表情显示** | 圆形屏幕动态表情 |

## 30 秒启动

已配置好环境？直接运行：

```bash
source ~/.bashrc
ros2 launch robot_bringup bringup.launch.py
```

首次配置？请查看 [快速开始](00_快速开始/快速开始.md)。
